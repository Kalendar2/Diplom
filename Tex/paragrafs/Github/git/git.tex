\texttt{sudo apt install git} –  устанавливает Git – систему контроля версий. Она нужна для отслеживания изменений в коде, совместной работы над проектами, клонирования репозиториев с GitHub.
 
 \texttt{sudo apt install nodejs} – устанавливает Node.js – среду выполнения JavaScript на сервере. Позволяет запускать скрипты на JavaScript на стороне сервера, работать с файлами, сетью и т.д.
 
 \texttt{sudo apt install npm} – устанавливает npm (Node Package Manager) – менеджер пакетов для Node.js. С его помощью можно устанавливать, обновлять и удалять сторонние библиотеки и инструменты для JavaScript-проектов.
 
 \texttt{sudo apt install grunt} – устанавливает Grunt – сборщик для автоматизации рутинных задач при разработке: минификация кода, компиляция, проверка синтаксиса, запуск тестов и т.д.

Чтобы проверить, что каждый компонент установлен корректно, можно проверить его версию. Например, для git используется команда \texttt{git --version}.
