Для работы с репозиторием проекта «Час ЕГЭ» сначала необходимо зарегистрироваться на платформе GitHub, перейдя по ссылке https://github.com/ и заполнив все требуемые поля (имя пользователя, адрес электронной почты, пароль) [6]. Регистрация нужна для того, чтобы в дальнейшем можно было клонировать репозиторий, а при необходимости — вносить свои изменения или отслеживать обновления.

Затем следует перейти по прямой ссылке на репозиторий проекта: https://github.com/nickkolok/chas-ege/. На открывшейся странице находится зелёная кнопка с надписью «Code» — при нажатии на неё появляется выпадающее меню, где показаны различные способы подключения. Необходимо скопировать ссылку для клонирования. Эту операцию рекомендуется выполнить сразу, пока компьютер не испытывает высокой нагрузки, так как работа с Ubuntu в VirtualBox может быть ресурсоёмкой.

Далее заходим в терминал и создаём папку на рабочем столе командой:
\quad \verb|mkdir <название_папки>|.

Убедиться, что папка создана, можно с помощью команды \texttt{ls} — в выводе будут перечислены все папки на рабочем столе, среди которых должна быть только что созданная. Затем заходим в эту папку командой: \quad \verb|cd <название_папки>|.
и клонируем репозиторий: \quad \verb|git clone <ссылка\_на\_репозиторий>|.

Добавляем ссылку на основной репозиторий проекта (upstream) с помощью команды:
\quad \verb|git remote add upstream <ссылка\_на\_репозиторий>|.

После этого убеждаемся, что upstream подключился, выполнив:
\newline \quad \texttt{git fetch upstream}

Собираем проект командой: \texttt{grunt}

Эту команду нужно выполнять именно в той папке, куда был склонирован репозиторий, иначе Grunt не найдёт файл конфигурации.

Переходим в папку \texttt{sh} с помощью команды: \texttt{cd dist/sh}

\newline \quad \texttt{open otladka.html}

В открывшемся окне браузера запускаем любой шаблон для проверки работоспособности сборки.
