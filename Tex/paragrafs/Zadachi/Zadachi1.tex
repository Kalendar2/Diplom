В начале генерируются целых числа a, b, c в диапазоне от 1 до 99 с помощью функции \texttt{sl(1,99)}, которая возвращает случайное целое число из заданного интервала. Переменные a, b, c служат числителями для будущих дробей. Если в функцию передать третье число, то она будет возвращать случайное целое число с шагом равным этому числу. Все диапозоны значений подбираются таким образом, чтобы соответсвовать исходным заданиям и дополнять их.

Далее создаётся массив \texttt{div = [10, 100]}, который хранит в себе возможные знаменатели. При генерации задания каждый раз случайным образом выбирается один из этих знаменателей с помощью метода \texttt{.iz()}. Это позволяет получать десятичные дроби с одним или двумя знаками после запятой.

\texttt{retryWhileError(function(){...}, 100000)} – это обёртка, которая повторяет генерацию задания до тех пор, пока не будет получен корректный вариант без ошибок (например, деления на ноль или некорректного формата). Число 100000 – максимальное количество попыток.

\texttt{'use strict'} – включение строгого режима JavaScript.

Выражение записывается после \texttt{expr:} и формируется с использованием нескольких встроенных функций и методов:

    \texttt{divideColon(...)} – функция, которая форматирует дробь в виде, удобном для текстового представления.

    \texttt{.shuffle()} – метод, перемешивающий элементы массива.

    \texttt{.iz()} – отвечает за выбор случайного знаменателя из массива.

    \texttt{join(['+','-'].iz())} – внешнее соединение между двумя частями выражения. Внутри указывается, какой ставится случайный знак + или -.
    
Вычисление ответа, берётся на себя конструкция  \texttt{NAtask.setEvaluationTask}, которая автоматически вычисляет ответ на основе переданного выражения.
