Переменные x и y независимо принимают значения от 2 до 10. Это коэффициенты подкоренного выражения.

\texttt{divider} — общий знаменатель для дробных частей чисел a и b.

\texttt{dividendA} и \texttt{dividendB} — числители для a и b.

Функция \texttt{genAssertIrreducible(dividend, divider)} гарантирует, что дробь \texttt{dividend/divider} не сокращается. Это важно, потому что в шаблоне дробь будет представлена в виде смешанного числа.

Для каждого из чисел a и b вычисляются:

    целая часть \texttt{whole = Math.floor(dividend / divider)}

    остаток (числитель дробной части) \texttt{num = dividend \% divider}

    знаменатель \texttt{den = divider}

Затем при подстановке в выражение используется функция \texttt{mixed(whole, num, den)}, которая форматирует число как смешанное. Альтернативно, может быть выбрана просто целая часть или обыкновенная дробь — в зависимости от результата вызова \texttt{.iz()}.

Поле \texttt{variables} используется для отдельных переменных, при значениях которых будет решаться сгенерированное задание.
