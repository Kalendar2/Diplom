Экспорт конфигурации виртуальной машины выполняется в файл формата .ova (Open Virtual Appliance). Это универсальный формат для хранения образа виртуальной машины, который поддерживается большинством систем виртуализации.

В меню программы нужно зайти в «Файл» и выбрать пункт «Экспорт конфигураций». В открывшемся окне выбираем машину для экспорта, и нажимаем «Далее».

Для экспорта в VirtualBox в главном меню программы необходимо открыть раздел «Файл» и выбрать пункт «Экспорт конфигураций». В появившемся окне нужно отметить виртуальную машину, которую нужно экспортировать, и нажать «Далее». На следующем шаге можно выбрать формат экспорта. По умолчанию предлагается «Open Virtualization Format 1.0», файлы .ova или .ovf.

После нажатия «Экспорт» начинается процесс упаковки. Он может занять несколько минут в зависимости от размера виртуального диска. В результате получается один файл с расширением .ova, который уже необходимо будет импортировать.
